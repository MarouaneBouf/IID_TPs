def devision(E) :
n = len(E)
E1,E2 = [],[]
k=0
while k < n//2 :
   E1.append(E[k])
   k+=1
while k < n :
   E2.append(E[k])
   k += 1
return E1,E2

La complexité de cet algorithme de cette fonction en big O notation est O(n).

L'opération principale de ce code est la création des deux listes E1 et E2, qui sont toutes deux faites dans une boucle qui parcourt les éléments de la liste d'entrée E. La boucle s'exécute pour n/2 itérations dans le premier cas, et n/2 itérations dans le second cas, pour un total de n itérations. Par conséquent, la complexité de temps de ce code est O(n).

Il est important de noter que la complexité de temps de ce code pourrait être améliorée en utilisant une seule boucle pour créer à la fois E1 et E2, plutôt que deux boucles séparées. Cela réduirait la complexité de temps à O(n/2), ou O(n) en notation de big O. Cependant, cette optimisation n'aurait pas d'impact sur l'ordre de grandeur global de la complexité de temps.

STRASSEN

  1. L'algorithme de Strassen est une méthode pour multiplier des matrices de manière plus efficace que la méthode classique de multiplication de matrices.
  1. Au lieu de multiplier chaque élément de l'une des matrices par chaque élément de l'autre matrice, l'algorithme de Strassen utilise des opérations de découpage, de sommation et de soustraction pour effectuer des multiplications partielles et obtenir le résultat final.
  1. Cet algorithme est plus rapide que la méthode classique lorsque les matrices sont de grande taille, car il nécessite moins de multiplications.
  1. Pour utiliser l'algorithme de Strassen, les matrices doivent être de taille 2^n x 2^n.
  1. Bien que l'algorithme de Strassen soit plus rapide que la méthode classique pour certaines tailles de matrices, il est plus compliqué à implémenter et peut nécessiter plus de mémoire.

def squareRoot(number, precision):

	start = 0
	end, ans = number, 1

	while (start <= end):
		mid = int((start + end) / 2)

		if mid**2 == number:
			ans = mid
			break

		if mid**2 < number:
			start = mid + 1
			ans = mid

		else:
			end = mid - 1

	increment = 0.1
	for _ in range(precision):
		while (ans * ans <= number):
			ans += increment
   
		ans = ans - increment
		increment = increment / 10

	return ans

#include <iostream>

struct Node {
  int data;
  Node* next;
};

// Fonction de recherche d'une valeur donnée dans la liste chaînée circulaire
bool search(Node* head, int value) {
  Node* curr = head;  // Initialiser le noeud courant à la tête

  // Vérifier si la liste est vide
  if (curr == nullptr) {
    std::cout << "La liste est vide" << std::endl;
    return false;
  }

  // Vérifier si la valeur se trouve dans le premier noeud
  if (curr->data == value) {
    return true;
  }

  // Parcourir la liste jusqu'à trouver la valeur ou atteindre le noeud de départ à nouveau
  curr = curr->next;
  while (curr != head) {
    if (curr->data == value) {
      return true;
    }
    curr = curr->next;
  }

  // Si la valeur n'a pas été trouvée, renvoyer false
  return false;
}

int main() {
  // Créer une liste chaînée circulaire avec 3 noeuds
  Node* head = new Node{1, nullptr};
  Node* second = new Node{2, nullptr};
  Node* third = new Node{3, nullptr};

  head->next = second;
  second->next = third;
  third->next = head;

  // Rechercher la valeur 2
  bool result = search(head, 2);
  std::cout << "Résultat : " << result << std::endl;  // Sortie : true

  // Rechercher la valeur 4
  result = search(head, 4);
  std::cout << "Résultat : " << result << std::endl;  // Sortie : false

  return 0;
}

import itertools
def sacADos_dynamique(capacite, elements):
    # Initialiser la matrice à 0
    matrice = [[0 for _ in range(capacite + 1)] for _ in range(len(elements) + 1)]

    # Remplir la matrice avec la méthode de programmation dynamique
    for i, w in itertools.product(range(1, len(elements) + 1), range(1, capacite + 1)):
        matrice[i][w] = (
            max(
                elements[i - 1][2] + matrice[i - 1][w - elements[i - 1][1]],
                matrice[i - 1][w],
            )
            if elements[i - 1][1] <= w
            else matrice[i - 1][w]
        )
    # Récupérer les éléments en fonction de la somme
    w = capacite
    n = len(elements)
    elements_selection = []

    while w >= 0 and n >= 0:
        e = elements[n-1]
        if matrice[n][w] == matrice[n-1][w-e[1]] + e[2]:
            # Ajouter l'élément courant à la liste des éléments sélectionnés
            elements_selection.append(e)
            w -= e[1]
        n -= 1
        
    # Renvoyer la valeur maximale et la liste des éléments sélectionnés
    return matrice[-1][-1], elements_selection

Tri par selection et insertion

La fonction tri_insertion(A) prend en entrée une liste A et effectue un tri par insertion de cette liste de manière récursive.

La fonction commence par vérifier si la liste A est vide ou ne contient qu'un élément. Si c'est le cas, la liste est déjà triée et la fonction retourne simplement la liste telle quelle.

Sinon, la fonction stocke le premier élément de la liste dans la variable valeur et crée une nouvelle liste reste avec les éléments restants. Elle appelle récursivement la fonction tri_insertion() sur la liste reste pour la trier, et stocke le résultat dans la variable reste_trié.

Ensuite, la fonction construit et retourne une nouvelle liste qui contient d'abord tous les éléments de reste_trié qui sont plus petits que valeur, puis valeur lui-même, et enfin tous les éléments de reste_trié qui sont plus grands ou égaux à valeur. Cette nouvelle liste est construite à l'aide de deux list comprehensions.

En résumé, la fonction tri_insertion() effectue un tri par insertion de la liste A de manière récursive en divisant la liste en deux parties : le premier élément et les éléments restants, et en insérant le premier élément à sa place finale dans la liste triée des éléments restants.

def tri_insertion(A):
    if len(A) <= 1: return A
    valeur = A[0]
    reste = A[1:]
    reste_trié = tri_insertion(reste)
    return [e for e in reste_trié if e < valeur] + [valeur] + [e for e in reste_trié if e >= valeur]

Q1 : Pour écrire la fonction fact(p) qui calcule la factorielle d'un entier positif p, on peut utiliser une approche récursive comme ceci :

// Fonction qui calcule la factorielle d'un entier positif
int fact(int p) {
    // Si p est 0 ou 1, la factorielle est 1
    if (p <= 1) {
        return 1;
    }
    // Sinon, la factorielle est p multipliée par la factorielle de p - 1
    else {
        return p * fact(p - 1);
    }
}

Cette fonction utilise un appel récursif pour calculer la factorielle de p. Si p est 0 ou 1, la factorielle est 1. Sinon, la factorielle est calculée en multipliant p par la factorielle de p - 1. La fonction s'appelle elle-même récursivement jusqu'à atteindre la base de cas où p est 0 ou 1, puis retourne le résultat du calcul.

Q2 : Pour écrire la fonction produit(n, k) qui calcule le produit des k premiers entiers inférieurs à n, on peut utiliser une approche récursive comme ceci :

// Fonction qui calcule le produit des k premiers entiers inférieurs à n
int produit(int n, int k) {
    // Si k est égal à 1, le produit est n
    if (k == 1) {
        return n;
    }
    // Sinon, le produit est n multiplié par le produit des k - 1 premiers entiers inférieurs à n - 1
    else {
        return n * produit(n - 1, k - 1);
    }
}

Q3 : Pour écrire la fonction binomial(n, k) qui calcule le coefficient binomial (n, k), on peut utiliser la formule suivante :

Copy code
(n, k) = n! / (k! * (n - k)!)

On peut donc écrire la fonction de cette façon :

// Fonction qui calcule le coefficient binomial (n, k)
int binomial(int n, int k) {
    return fact(n) / (fact(k) * fact(n - k));
}

Notez que la fonction fact(p) qui calcule la factorielle d'un entier positif p a été définie précédemment.
